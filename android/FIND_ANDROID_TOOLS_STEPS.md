# Пошаговая инструкция: Где найти Android Tools

## Секции, где НЕ искать:
- ❌ **Build Tools** - там только компиляторы (MinGW, CMake, Ninja)
- ❌ **Qt 6.7.2 → Android** - там только библиотеки Qt для Android

## Секции, где ИСКАТЬ:

### 1. Developer and Designer Tools
Прокрутите список **выше** секции "Build Tools" и ищите:
```
Developer and Designer Tools
  ├── Qt Creator
  ├── Qt Designer
  ├── Qt Linguist
  └── Android Tools  ← ВОТ ОНО!
```

### 2. Tools (отдельная секция)
Может быть отдельная секция:
```
Tools
  └── Android Tools  ← ВОТ ОНО!
```

### 3. Используйте поиск
1. В поле поиска (с иконкой лупы) введите: **"Android Tools"**
2. Или попробуйте: **"androiddeployqt"**
3. Должен появиться компонент

## Что делать, если не нашли:

### Вариант 1: Прокрутите весь список
1. Прокрутите список компонентов **от начала до конца**
2. Ищите любую секцию, содержащую слово "Tools" или "Android"

### Вариант 2: Проверьте фильтры
1. В выпадающем меню "Show" выберите **"All"** (если не выбрано)
2. Это покажет все доступные компоненты, включая не установленные

### Вариант 3: Android Tools может называться по-другому
Попробуйте поискать:
- "Qt Android Tools"
- "Android SDK Tools"
- "androiddeployqt"

## Важно:

Android Tools - это **отдельный компонент**, который устанавливается независимо от Qt 6.7.2. Он нужен для создания APK файлов.

Если вы не можете найти Android Tools, возможно:
1. Он не доступен для вашей версии Qt
2. Нужно обновить Qt Maintenance Tool
3. Или использовать альтернативный способ создания APK
