  Voice Upside Down
═══════════════════════════════════════════════════════════════

┌─────────────────────────────────────────────────────────────┐
│ БЫСТРЫЙ СТАРТ (автоматический выбор)                       │
└─────────────────────────────────────────────────────────────┘

Просто запустите: installer\create_installer.bat

Скрипт автоматически определит доступный инструмент 
(Qt Installer Framework или Inno Setup) и создаст установщик!


┌─────────────────────────────────────────────────────────────┐
│ РУЧНОЙ ВЫБОР                                                │
└─────────────────────────────────────────────────────────────┘

Есть два варианта создания установщика:

┌─────────────────────────────────────────────────────────────┐
│ ВАРИАНТ 1: Qt Installer Framework (рекомендуется)          │
└─────────────────────────────────────────────────────────────┘

Требования:
  • Qt 5.15.2 с установленным Qt Installer Framework
  • Собранное приложение в Release режиме

Шаги:
  1. Запустите: installer\prepare_installer.bat
     (укажите пути к Qt и сборке, если отличаются от стандартных)
  
  2. Запустите: installer\build_installer.bat
     (укажите путь к Qt Installer Framework, если отличается)
  
  3. Готово! Файл VoiceUpsideDown_Installer.exe создан в корне проекта


┌─────────────────────────────────────────────────────────────┐
│ ВАРИАНТ 2: Inno Setup (проще, не требует Qt IFW)          │
└─────────────────────────────────────────────────────────────┘

Требования:
  • Inno Setup Compiler (скачать: https://jrsoftware.org/isdl.php)
  • Qt 5.15.2 с windeployqt
  • Собранное приложение в Release режиме

Шаги:
  1. Запустите: installer\prepare_for_inno.bat
     (укажите пути к Qt и сборке, если отличаются от стандартных)
  
  2. Откройте файл installer\VoiceUpsideDown.iss в Inno Setup Compiler
     Нажмите Build -> Compile (или F9)
  
  3. Готово! Файл VoiceUpsideDown_Setup.exe создан в корне проекта


═══════════════════════════════════════════════════════════════
  ЧТО ВКЛЮЧАЕТ УСТАНОВЩИК
═══════════════════════════════════════════════════════════════

Установщик автоматически включает:
  ✓ Исполняемый файл voice_upside_down.exe
  ✓ Все необходимые Qt DLL (Qt5Core, Qt5Gui, Qt5Qml, Qt5Quick, Qt5Multimedia)
  ✓ QML модули и плагины
  ✓ Visual C++ Runtime DLL (если найдены)
  ✓ Все ресурсы приложения

Приложение будет работать на "голом компьютере" без установленного Qt!
