# Мастерская голоса (Voice Upside Down)

Десктопное приложение для работы с аудио-отрезками: запись, воспроизведение и реверс аудио-сегментов.

## Возможности

- **Загрузка аудио-файлов**: Поддержка MP3 и WAV файлов
- **Запись с микрофона**: Прямая запись аудио с микрофона
- **Автоматическая сегментация**: Автоматическое разделение аудио на отрезки (настраиваемая длина, 1-10 секунд)
- **Управление отрезками**: 
  - Запись голоса для каждого отрезка
  - Воспроизведение оригинального отрезка
  - Воспроизведение записанного отрезка
  - Воспроизведение реверса оригинального отрезка
- **Обработка аудио**:
  - Склейка всех записанных отрезков в один WAV файл
  - Реверс всей склеенной песни
  - Воспроизведение склеенной и реверсной версий
- **Управление проектами**: Сохранение и загрузка проектов (JSON формат с путями к аудио-файлам и параметрами отрезков)

## Требования

- Qt 5.15 или новее
- Компилятор с поддержкой C++17
- CMake 3.16 или новее

## Сборка

```bash
mkdir build
cd build
cmake ..
cmake --build .
```

## Использование

1. **Загрузка аудио**: Нажмите "Загрузить файл" для загрузки MP3 или WAV файла, или нажмите "Включить микрофон" для записи с микрофона
2. **Настройка длины отрезка**: Используйте поле "Длина отрезка" для установки длины отрезка (1-10 секунд)
3. **Работа с отрезками**: 
   - Запишите голос для каждого отрезка, используя кнопку "Записать"
   - Воспроизведите оригинальный отрезок, используя "Воспроизвести оригинал"
   - Воспроизведите записанный отрезок, используя "Воспроизвести запись"
   - Воспроизведите реверс оригинального отрезка, используя "Реверс оригинал"
4. **Обработка аудио**:
   - Нажмите "Склеить отрезки" для объединения всех записанных отрезков
   - Нажмите "Перевернуть песню" для реверса склеенной песни
   - Используйте кнопки воспроизведения для прослушивания результатов
5. **Сохранение/Загрузка проектов**: Используйте "Сохранить проект" и "Открыть проект" для сохранения и загрузки вашей работы

## Технические детали

- **Фреймворк**: Qt 5.15 или Qt 6.7+ (QML + C++)
- **Платформы**: Windows (Desktop), Android (arm64-v8a)
- **Декодирование аудио**: Собственный MP3 декодер (minimp3) и парсер WAV
- **Воспроизведение аудио**: QAudioOutput (Qt 5) / QAudioSink (Qt 6)
- **Запись аудио**: QAudioInput (Qt 5) / QAudioSource (Qt 6)
- **Формат проекта**: `.vups` файл (JSON) с директорией проекта, содержащей все аудио-файлы
- **Сохранение проектов**: 
  - При сохранении создается файл `.vups` и директория проекта
  - Если запись была с микрофона, сохраняется файл `[имя]_song.wav`
  - Если есть склеенная песня, сохраняется `[имя]_fragment_song.wav` и `[имя]_revers_fragment_song.wav`
